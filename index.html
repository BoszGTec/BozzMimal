Usage

Just Insert Link Tag

Quick start

add html tag<link rel="stylesheet" href="{link}">in html.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>BozzMimal</title>
    <link rel="stylesheet" href="./BozzMimal.css">
  </head>
  <body>
    <h1>Hello Ha...</h1>
  </body>
</html>
